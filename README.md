# Simulador de Máquina de Turing

Simulador de Máquina de Turing desenvolvido em Python para a disciplina de Linguagens Formais e Autômatos.

## Funcionalidades

- Simulação de Máquinas de Turing determinísticas
- Leitura de definições de MT a partir de arquivos
- Execução passo a passo
- Visualização do estado da fita

## Arquivos de Exemplo

- `palindromo.txt` - MT para reconhecimento de palíndromos
- `soma.txt` - MT para operação de soma unária

## Como Executar

```bash
python simturing.py <arquivo_mt>
```

## Formato do Arquivo

O arquivo de entrada deve conter a especificação da MT conforme descrito em `2o trabalho - simulador MT.pdf`.

## Requisitos

- Python 3.x
